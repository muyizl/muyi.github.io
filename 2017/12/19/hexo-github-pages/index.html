域名绑定

购买域名

建议大家去 GoDaddy购买，第一年很便宜。

申请免费域名

.ML免费域名申请注册、 .CF免费域名申请注册、 .GA免费域名申请注册、 .TK免费域名申请注册

申请域名都比较简单，实在不会就Google/百度一下，很多教程，TK域名算是一个老牌的域名了，据说目前活跃的域名超过1600万个，这里估计国人也贡献了不少。

将自己的域名和Github Pages绑定，需要做以下几件事情：

修改域名DNS

详情：修改你的域名DNS为f1g1ns1.dnspod.net 和 f1g1ns2.dnspod.net 。

创建CNAME文件

详情：在你的本站点目录的source目录下添加一个CNAME文件，以文本编辑器打开，里面添加你的域名信息（不带有http），例如：baidu.com 。

配置DNSPOD

详情：前往DNSPOD官网注册，并添加域名，添加成功后点击域名链接，进入域名记录界面添加三条如下记录：
